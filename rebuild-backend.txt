Now let's rebuild the backend with all the fixes:

Run this command to rebuild just the backend:

ssh root@5.78.147.82 "cd /var/www/Haveninstitute && docker compose stop backend && docker compose build backend && docker compose up -d backend"

Then wait 30 seconds and test:

ssh root@5.78.147.82 "cd /var/www/Haveninstitute && sleep 30 && curl http://localhost:3001/api/v1/health"

If the backend health check passes, your website should be working!
